## 1. 获取认证凭证

根据你的使用场景，选择合适的认证方式：

<Tabs>
  <Tab title="API Key（推荐）">
    适合第三方开发者、机器人客户端。

    <Steps>
      <Step title="注册账号">
        前往 [协议终端](https://end.shallow.ink/login) 注册并登录。
      </Step>
      <Step title="创建 API Key">
        进入 [开发者 → API 密钥](https://end.shallow.ink/dashboard/developer/apikeys) 页面，点击「创建密钥」。
      </Step>
      <Step title="保存密钥">
        复制生成的 API Key（格式：`sk_xxxxxxxxxxxx`）。

        <Warning>
          API Key 仅在创建时显示一次，请妥善保存。
        </Warning>
      </Step>
    </Steps>
  </Tab>
  <Tab title="Anonymous Token">
    适合前端应用的游客模式，无需注册。

    ```bash
    curl -X POST https://api.end.shallow.ink/api/v1/auth/anonymous-token \
      -H "Content-Type: application/json" \
      -d '{"fingerprint": "your-device-fingerprint-32chars-min"}'
    ```

    响应：
    ```json
    {
      "code": 0,
      "message": "成功",
      "data": {
        "token": "anon_xxxxxxxxxxxxxxxx",
        "expires_in": 7200
      }
    }
    ```
  </Tab>
</Tabs>

## 2. 发送第一个请求

使用你的凭证调用健康检查接口，验证连接：

<CodeGroup>
```bash cURL
curl https://api.end.shallow.ink/health
```

```javascript JavaScript
const response = await fetch('https://api.end.shallow.ink/health');
const data = await response.json();
console.log(data);
```

```python Python
import requests

response = requests.get('https://api.end.shallow.ink/health')
print(response.json())
```

```go Go
resp, err := http.Get("https://api.end.shallow.ink/health")
```
</CodeGroup>

响应：

```json
{
  "code": 0,
  "message": "成功",
  "data": {
    "status": "healthy",
    "timestamp": "2026-01-26T14:00:00+08:00",
    "uptime": 123.45,
    "memory": {
      "alloc_mb": 5,
      "total_alloc_mb": 10,
      "sys_mb": 15
    },
    "runtime": {
      "version": "go1.21.0",
      "goroutines": 10,
      "os": "windows",
      "arch": "amd64"
    }
  }
}
```

## 3. 查询 Wiki 数据

使用认证凭证查询游戏百科数据：

<CodeGroup>
```bash API Key
curl https://api.end.shallow.ink/api/wiki/categories \
  -H "X-API-Key: sk_your_api_key"
```

```bash Anonymous Token
curl https://api.end.shallow.ink/api/wiki/categories \
  -H "X-Anonymous-Token: anon_your_token"
```

```javascript JavaScript
const response = await fetch('https://api.end.shallow.ink/api/wiki/categories', {
  headers: {
    'X-API-Key': 'sk_your_api_key'
  }
});
const data = await response.json();
console.log(data.data);
```
</CodeGroup>

## 4. 查询游戏数据（需要 Framework Token）

游戏数据查询需要额外的 **Framework Token**（通过森空岛登录获取）：

```bash
curl https://api.end.shallow.ink/api/endfield/user \
  -H "X-API-Key: sk_your_api_key" \
  -H "X-Framework-Token: your_framework_token"
```

<Note>
  Framework Token 是游戏数据查询凭证，通过扫码登录或手机登录获取。接口认证（API Key / JWT / Anonymous Token）和游戏数据凭证（Framework Token）是独立的两层。
</Note>
